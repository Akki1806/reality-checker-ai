# 🎯 Reality Checker AI

An AI-powered tool that evaluates how realistic your goals actually are.

> Everyone online says “you can achieve anything.”  
> This tool tells you the truth.

---

## 🚀 Live Demo
👉 https://goal-reality-check--akashpushpakap.replit.app

---

## 🧠 What it does

- Generates a **Reality Score (0–100)**
- Classifies goals as **Realistic / Risky / Unrealistic**
- Highlights key risk factors
- Provides actionable plans

---

## 💡 Example

Goal: Earn 1 crore in 6 months from trading  
→ Score: 8/100  
→ Unrealistic
